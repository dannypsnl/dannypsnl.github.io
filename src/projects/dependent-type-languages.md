The followings are dependent type languages I try to create.

- [semi](https://github.com/dannypsnl/semi) macro as type for dependent type theory work, by using `local-expand` and `syntax-property`
- [typical](https://github.com/dannypsnl/typical) try macro as type and learn how to do termination check
- [incr](https://github.com/dannypsnl/incr) inductive based on introduction/elimination rules
- [inductive](https://github.com/dannypsnl/inductive) try to build inductive language
